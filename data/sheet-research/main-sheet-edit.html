Decision to cast skills for each turn in battle
- Search for skill from 1st slot to last slot:
+ check if skill is useable:
* to decide if skill useable: it checks skill's category, skill's type and skill's weapons:
If if is activity skill category, then nope.
If not enough MP then nope.
If it is attack skill and skill has weapon's type then check equipped weapon, it it doesn't match then nope. AA3 doesn't weapon's type so Bow or not doesn't matter.
If it is recovery skill then check current user's HP if it maxed then nope.
If it is revival skill then check unit's HP, if it is bigger than 0 then nope.
if skill is invoking then nope
if it is neither Line Attack, Area Attack or Debuff then YES
else if it is in front row: check if there is any one in skill's range (skill's range from Skill tab), if noone then nope, else YES
else: NOPE
+ if skill is useable then throw dice to know if it can be casted, if not, go to next slot <= slot in first position got better chance to be casted than last position

So only Line Attack, Area Attack and Debuff need to be in front line, any thing else could be done from range

81
unit's speed decideds the unit's attack frame, which is around 30 * (speed / 25) ^ (-0.362), the higher speed is, the lower attack frame will be, so unit will do action faster
