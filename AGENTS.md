\# Project Instructions



\## Real app location

The actual website is inside:



artifacts/marriage-matcher



\## Local run (Windows)



Use these commands:



cd artifacts\\marriage-matcher

set PORT=5173

set BASE\_PATH=/

npm run dev -- --host



\## Important



\- Do not try to run from repo root

\- Do not waste turns guessing preview commands

\- Use the commands above first

\- For phone testing, use LAN IP with port 5173

\- Prefer fixing files over repeatedly restarting preview
